# KPI Definitions

| KPI | Formula |
|---|---|
| Total Revenue | Sum of actual revenue |
| Revenue Variance | Actual revenue - budget revenue |
| Revenue Variance % | Revenue variance / budget revenue |
| COGS | Cost of goods sold required to deliver revenue |
| Gross Profit | Revenue - COGS |
| Gross Margin % | Gross profit / revenue |
| Operating Expenses | Payroll + other operating expenses |
| Expense Variance | Actual operating expense - budget operating expense |
| Expense Variance % | Expense variance / budget operating expense |
| EBITDA / Operating Profit | Gross profit - operating expenses |
| EBITDA Margin % | EBITDA / revenue |
| Department Overspend | Department where actual expense exceeds budget |
| Unfavorable Variance Driver | Revenue shortfall or expense overspend with negative profit impact |
